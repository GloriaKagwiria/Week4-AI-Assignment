### Question 2
#### Explain the difference between:
- Structured Data
- Unstructured Data
##### Give two Examples of each.
1. `Structured data` has fixed schema and is organized in tables.**Examples** include *phone numbers* and *names*.
2. `Unstructured data` has no fixed schema and is stored in document files or data lakes.**Examples** include *photos* and *videos*.
   
### Question 3
#### What is a  Schema?
##### Explain using a simple example.
Schema is like a *blueprint* that defines how data is `stored` and `related in a database` based on the *nature* of the given data.
- Examples
1. Phone number-consists of numbers only.
2. Name-consists of letters only.
3. Photos and videos-can't be certain what kind of data types is contained in side them.
- Therfore phone number and names haves `fixed schema` and can be organized in a database whereas `photos and videos` have no fixed schema and cannot be easily organized.
